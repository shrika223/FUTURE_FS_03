# FUTURE_FS_03

## 📖 Overview
FUTURE_FS_03 is a **full‑stack café website** built using **React (frontend)**, **Node.js + Express (backend)**, and **MongoDB (database)**.  
It demonstrates a recruiter‑ready project with a menu, ordering system, and database integration.

---

## 🎯 Objectives
- Build a professional full‑stack project.  
- Implement a café menu and ordering system.  
- Connect frontend and backend with REST APIs.  
- Use MongoDB for storing menu items and orders.  

---

## 🛠️ Tech Stack
- **Frontend:** React, React Router, CSS  
- **Backend:** Node.js, Express  
- **Database:** MongoDB, Mongoose  
- **Tools:** Git, GitHub, VS Code
